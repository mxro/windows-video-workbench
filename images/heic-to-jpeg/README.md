# HEIC to JPEG Converter

Converts HEIC/HEIF images to JPEG.

## Usage

**Option 1:** Drag and drop HEIC/HEIF files onto `heic-to-jpeg.bat`

**Option 2:** Run the script in a folder containing HEIC/HEIF files

## Output

JPEG files are saved in the same folder as the source images with the original filename.
