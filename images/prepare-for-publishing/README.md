# Prepare for Publishing

Converts various image formats to JPEG with medium quality and removes metadata.

## Usage

**Option 1:** Drag and drop image files onto `prepare-for-publishing.bat`

**Option 2:** Run the script in a folder containing image files

## Output

JPEG files are saved in the same folder as the source images with a `.pub` suffix (e.g., `photo.jpg` → `photo.pub.jpg`)
